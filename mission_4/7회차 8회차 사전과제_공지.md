## 왜 7,8회차 과제가 도전적인가?  
  
### 1. 새로운 도구를 익혀야 합니다  
  
디스포저블 앱 제작을 위해 AI 기반 앱 빌더를 활용해야 합니다.  
- **Nano Banana Pro**: 자연어로 웹 앱 즉석 생성  
- **Claude Artifacts**: Claude 내장 앱 생성 기능  
- **v0.dev**: Vercel의 UI 생성 도구  
  
### 2. 프롬프트를 통한 앱 설계 능력이 필요합니다  
  
코딩 없이 자연어로 앱을 만들기 위해서는 **명확한 요구사항 정의**가 필수입니다.  
- 어떤 입력을 받을 것인가?  
- 어떤 출력을 내보낼 것인가?  
- 사용자 인터페이스는 어떻게 구성할 것인가?  
  
### 3. 30분 이내 제작이라는 시간 제약이 있습니다  
  
디스포저블 앱의 핵심은 **'빠르게 만들고 빠르게 버리는 것'**입니다.  
- 완벽주의를 버리고 "작동하면 OK"의 마인드셋 필요  
- 처음부터 복잡한 기능을 넣지 않고 핵심 기능만 구현  
- 시간 초과 시에도 실제 소요 시간을 정직하게 기록
