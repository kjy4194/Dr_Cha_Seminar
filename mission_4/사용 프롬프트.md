다음 요구사항으로 단일 HTML 파일로도 동작하는 React 웹 앱을 만들어 주세요.
(Artifacts로 실행 가능하게, 외부 API 호출 없이)
[앱 이름] 글로벌 회의 시간 변환기

[기능]

1. 날짜: yyyy-mm-dd 입력 필드 (기본값: 오늘)

2. 시각: HH:mm 입력 (24시간제)

3. 기준 타임존: 드롭다운 — 최소한 "Asia/Seoul", "UTC", "America/New_York" 포함

4. "변환" 버튼 클릭 시, 선택한 기준 날짜·시각을 해당 타임존의 로컬 시각으로 해석한 뒤,

   아래 타임존 각각에 대해 **같은 순간**의 날짜·시각을 표로 표시:

   - Asia/Seoul

   - UTC

   - America/New_York

   - Europe/Berlin

5. 타임존 이름 옆에 약어 표기도 (예: KST, UTC, ET는 대략 표기)

6. 잘못된 날짜/시각이면 빨간색으로 에러 메시지

7. UI는 깔끔한 카드 레이아웃, 반응형
  
[기술]

- React (함수 컴포넌트)

- 시간대 변환은 브라우저 내장 Intl 또는 luxon/dayjs 중 하나로 일관되게 처리

- 한국어 UI 라벨 사용 (날짜, 시각, 기준 타임존, 변환 결과)
