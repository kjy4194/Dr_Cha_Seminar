## 1. T자형 역량 이해하기

### 1.1 T자형 역량이란?

T자형 역량은 **깊은 전문성(세로축)과 넓은 기본 역량(가로축)을 갖춘 인재상**을 의미합니다.

```
                │
      ──────────┼──────────  ← 넓은 범위의 기본 역량 (가로)
                │
                │
                │            ← 깊은 전문성 (세로)
                │
                │
```

#### 세로축 (Vertical): 핵심 전문 분야
- 본인의 주요 전공 또는 경력 분야
- 깊이 있는 전문 지식과 경험
- 예: 로봇 제어, 재료 공학, 데이터 분석 등

#### 가로축 (Horizontal): AI로 확장한 영역
- 세미나를 통해 습득한 AI 도구 활용 능력
- 다양한 분야에 적용 가능한 기본 역량
- 예: 문서 작성, 코딩, 데이터 분석, 자동화 등

### 1.2 T자형 역량의 가치

| 기존 전문가 | T자형 역량 보유자 |
|-----------|-----------------|
| 한 분야만 깊음 | 한 분야는 깊고, 다른 분야도 기본 이상 |
| 외주 의존도 높음 | AI 도구로 자체 해결 가능 |
| 변화에 취약 | 다양한 분야 적응 가능 |
| 업무 효율성 낮음 | AI 활용으로 생산성 극대화 |

### 1.3 역할 유동성 (Role Fluidity)

역할 유동성은 **상황과 필요에 따라 다양한 역할을 유연하게 수행할 수 있는 능력**입니다.

```
┌─────────────────────────────────────────────────────────────┐
│  역할 유동성의 핵심                                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  "한 사람이 여러 역할을 상황에 맞게 수행"                    │
│                                                             │
│  예시:                                                      │
│  - 엔지니어 → 문서 작성자 (AI 활용)                         │
│  - 연구자 → 프로그래머 (자동화 도구)                        │
│  - 관리자 → 데이터 분석가 (AI 분석)                         │
│                                                             │
│  장점:                                                      │
│  ✓ 팀 내 유연한 역할 분담                                   │
│  ✓ 프로젝트 효율성 증대                                     │
│  ✓ 개인 역량 다각화                                         │
│  ✓ 조직 생산성 향상                                         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## ❓ 자주 묻는 질문 (FAQ)

### Q1: T자형 역량과 일자형 역량의 차이는?
**A**: 
- **T자형**: 한 분야는 깊고(세로), 다른 분야도 기본 이상(가로)
- **일자형(I자형)**: 한 분야만 매우 깊음
- **역T자형**: 여러 분야를 얕게 알지만 깊이가 없음

T자형이 현대 조직에서 가장 가치 있는 인재상입니다.

### Q2: 역할 유동성이 높으면 전문성이 떨어지지 않나요?
**A**: 아닙니다. T자형 역량은 전문성을 유지하면서 다른 분야도 기본 이상으로 수행할 수 있는 능력입니다. 세로축(전문성)은 깊게 유지하고, 가로축(기본 역량)을 확장하는 것입니다.

### Q3: 자가 진단표를 어떻게 객관적으로 작성하나요?
**A**: 
- 구체적인 사례를 바탕으로 작성
- 동료나 상사의 피드백 참고
- 실제 업무 성과 기준으로 평가
- 과장하지 않고 현실적으로 작성

### Q4: 향후 학습 계획이 너무 많으면 어떻게 하나요?
**A**: 우선순위를 정하세요:
1. 즉시 필요한 것 (단기)
2. 중요하지만 시간이 걸리는 것 (중기)
3. 장기적 관심사 (장기)

한 번에 하나씩 집중하는 것이 효과적입니다.

### Q5: 세미나 내용을 모두 기억하지 못했어요.
**A**: 괜찮습니다. 다음을 활용하세요:
