## 업무 분석

### 1. 업무명
- 일일 업무 보고서 작성
- 매일 퇴근 전 작성하는 일일 업무 리포트를 Agent가 자동으로 생성합니다.

### 2. 현재 수행 방식
- 소요 시간: 매일 10
- 반복 주기: 매일 1회
- 수작업 내용: 일일 업무 정리, 내일 계획 작성
- mkdir daily-report-agent
- cd daily-report-agent

### 3. 입력 정보
- 오늘 완료한 업무 목록
- 내일 예정 업무
- 특이사항/이슈
- 위의 내용으로 일일 업무 보고서 작성해주세요.

### 4. 출력 결과물
- 일일 업무보고서.md 파일
- 결과를 daily_report_[오늘날짜].md 파일로 저장
- 형식: 마크다운 표 형식
- 제목: 일일업무보고서 (날짜)
- 작성자: 김정엽
- 금일 업무 내용
- 명일 예정 업무 

### 5. 자동화 가능 부분
- 보고서 템플릿 생성
- 형식 통일 및 정리

### 6. 사람이 해야 할 부분
- 업무 내용 입력
- 최종 검토 및 수정

# Agent화 설계 문서

## 1. 프로젝트 개요

### 프로젝트명
일일 업무보고서 자동 생성 Agent

### 목표
입력된 업무 정보를 바탕으로 형식화된 일일업무보고서 자동 생성

### 예상 효과
- 기존 소요 시간: 10분
- 자동화 후 예상 시간: 1분
- 절감 시간: 9분/일일

---

## 2. Agent 설계

### 입력 (Input)
이번 주 완료 업무 (텍스트)
다음 주 예정 업무 (텍스트)
특이사항 (선택)

### 처리 과정 (Process)
```
1. 입력 정보 파싱
2. 보고서 템플릿 적용
3. 형식 정리 및 표 생성
4. 파일 저장
```

### 출력 (Output)
```
- 파일명: 주간업무보고서_YYYYMMDD.md
- 형식: 마크다운
```
## 3. 프롬프트 설계

### Agent 지시 프롬프트
```
[역할]
당신은 일일 업무 보고서 작성을 도와주는 AI 비서입니다.

[맥락]
사용자가 입력한 업무 정보를 바탕으로 일일 업무보고서를 작성합니다.
보고서는 회사의 표준 양식에 맞춰야 합니다.

[지시]
1. 사용자에게 오늘늘 완료 업무를 물어보세요
2. 내일 예정 업무를 물어보세요
3. 특이사항이 있는지 확인하세요
4. 입력받은 정보로 보고서를 작성하세요
5. 결과를 파일로 저장하세요

[출력 형식]
마크다운 형식, 표 사용, A4 1페이지 분량
```
