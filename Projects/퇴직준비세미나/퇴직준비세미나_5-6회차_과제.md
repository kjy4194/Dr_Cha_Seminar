# 퇴직준비세미나 5-6회차 과제

## 개요
- **과제명**: Obsidian PARA 구조 구축
- **제출 기한**: 2026-03-14
- **상태**: 진행 중 🔄

## 과제 목표
1. Obsidian vault에 PARA 폴더 구조 구축
2. 노트 12개 이상 작성
3. 양방향 링크([[wikilink]]) 10건 이상 연결
4. 그래프 뷰로 지식 네트워크 시각화

## PARA 구조 이해
PARA는 4개 카테고리로 지식을 관리하는 방법론이다.
→ 자세한 내용: [[PARA_방법론_완벽정리]]

## Obsidian 활용
노트 작성 및 링크 연결에 Obsidian을 사용한다.
→ 사용법 참고: [[Obsidian_활용법_정리]]

## 진행 상황
- [x] PARA 폴더 구조 생성 (Projects, Areas, Resourses, Archive)
- [x] Projects 노트 4개 작성
- [x] Areas 노트 4개 작성
- [x] Resourses 노트 4개 작성
- [x] Archive 노트 4개 작성
- [x] 양방향 링크 12건 이상 연결
