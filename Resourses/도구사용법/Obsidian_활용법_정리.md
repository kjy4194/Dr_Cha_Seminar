# Obsidian 활용법 정리

## 개요
Obsidian 핵심 기능, 단축키, 플러그인, PARA 운영 방법 총정리

## 기본 단축키

| 단축키 | 기능 |
|--------|------|
| `Ctrl+N` | 새 노트 생성 |
| `Ctrl+O` | 노트 빠른 열기 |
| `Ctrl+P` | 커맨드 팔레트 |
| `Ctrl+G` | 그래프 뷰 열기 |
| `Ctrl+E` | 편집/미리보기 전환 |
| `Ctrl+K` | 링크 삽입 |
| `Ctrl+B` | 굵게 |
| `Ctrl+I` | 기울임 |
| `[[` | 내부 링크 시작 (wikilink) |
| `![[` | 파일 임베드 |

## 핵심 기능

### 양방향 링크 (Bidirectional Links)
```markdown
[[노트이름]]           # 기본 링크
[[노트이름|표시할텍스트]]  # 별명 링크
[[노트이름#섹션]]       # 특정 섹션 링크
```

### 그래프 뷰 (Graph View)
- `Ctrl+G`: 전체 그래프 열기
- 노드 클릭: 해당 노트로 이동
- 필터로 특정 노트/태그만 표시 가능
- 지식 네트워크 시각화에 핵심 도구

### 백링크 패널
- 오른쪽 사이드바 → 백링크
- 현재 노트를 참조하는 모든 노트 표시
- [[양방향]] 특성 확인 가능

## 권장 플러그인 (Community)

| 플러그인 | 용도 |
|---------|------|
| Dataview | 노트를 데이터베이스처럼 쿼리 |
| Calendar | 일일 노트 달력 뷰 |
| Templater | 고급 템플릿 기능 |
| Git | GitHub 자동 백업 |
| Excalidraw | 다이어그램 작성 |

## PARA 시스템 운영법
PARA의 각 카테고리와 Obsidian 폴더를 1:1 매핑
→ 이론 자세히: [[PARA_방법론_완벽정리]]

### 빠른 캡처 워크플로우
1. `Ctrl+N` → 빠른 메모 작성
2. 필요 시 태그 추가 (#inbox)
3. 주간 리뷰 시 적절한 PARA 카테고리로 이동
