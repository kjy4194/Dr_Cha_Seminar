# Claude Code 명령어 모음

## 개요
Claude Code CLI의 주요 명령어 및 활용 패턴 정리

## 기본 명령어

| 명령어 | 설명 |
|--------|------|
| `claude` | Claude Code 시작 |
| `claude --help` | 도움말 보기 |
| `/help` | 대화 중 도움말 |
| `/clear` | 대화 내용 초기화 |
| `/exit` | Claude Code 종료 |
| `/status` | 현재 상태 확인 |

## 파일 관련 명령어

```bash
# 파일 읽기 요청
"파일 [경로]를 읽어줘"
"[파일명] 내용을 분석해줘"

# 파일 생성/수정
"[파일명] 파일을 만들어줘"
"[파일명]에서 [부분]을 [내용]으로 수정해줘"

# 디렉토리 탐색
"현재 폴더 구조를 보여줘"
"[폴더명] 안에 어떤 파일이 있어?"
```

## Git 관련 작업

```bash
# Git 상태 확인
"git 상태를 확인해줘"

# 커밋
"변경사항을 '[커밋 메시지]'로 커밋해줘"

# 브랜치
"새 브랜치 [브랜치명]을 만들어줘"
"현재 브랜치에서 main으로 merge해줘"
```

## 코드 생성 패턴

```python
# Python 스크립트 생성 요청 예시
"다음 기능을 하는 Python 스크립트를 작성해줘:
- CSV 파일을 읽어서
- 특정 컬럼의 통계를 계산하고
- 결과를 Excel로 저장"
```

## 일괄 작업 (배치 처리)

```bash
# 여러 파일 동시 생성
"다음 12개 파일을 PARA 구조에 맞게 생성해줘:
[파일 목록]"

# 파일 내용 검색
"[키워드]를 포함하는 파일을 모두 찾아줘"
```

## 유용한 프롬프트 패턴

### 분석 요청
```
"[파일/코드]를 분석하고 개선점을 제안해줘"
```

### 문서화
```
"이 코드에 한국어 주석을 추가해줘"
"README.md를 작성해줘"
```

### 디버깅
```
"이 오류를 해결해줘: [오류 메시지]"
"코드가 왜 작동하지 않는지 분석해줘"
```

## 고급 기능

### MCP (Model Context Protocol)
Claude Code는 외부 서비스와 연동 가능한 MCP 서버를 지원
→ 설정: `.claude/settings.json`의 mcpServers 섹션

### Hooks
특정 이벤트 발생 시 자동 실행되는 쉘 명령어 설정
→ 설정: `settings.json`의 hooks 섹션

### Permission Mode
- `default`: 위험한 작업은 확인 요청
- `auto-approve`: 자동 승인 (주의 필요)
