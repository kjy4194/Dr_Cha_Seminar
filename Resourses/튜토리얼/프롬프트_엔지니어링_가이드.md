# 프롬프트 엔지니어링 가이드

## 개요
AI 모델에게 효과적인 지시문을 작성하는 기술 모음

## RCIF 골든공식

| 요소 | 영문 | 설명 | 예시 |
|------|------|------|------|
| R | Role | AI에게 역할 부여 | "당신은 산업공학 전문가입니다" |
| C | Context | 배경 정보 제공 | "KIMM 연구소에서 보고서를 작성 중입니다" |
| I | Instruction | 구체적 지시 | "다음 데이터를 분석하여..." |
| F | Format | 출력 형식 지정 | "표 형식으로 3가지 항목을 정리해줘" |

## 프롬프트 템플릿 모음

### 연구 보고서 작성
```
Role: 당신은 산업기계 분야 전문 기술 작가입니다.
Context: KIMM(한국기계연구원) 산업기계DX연구실 연구원이
         [주제] 관련 기술보고서를 작성 중입니다.
Instruction: 다음 내용을 바탕으로 서론 섹션을 작성해주세요:
             - 연구 배경: [배경]
             - 연구 목적: [목적]
Format: A4 2페이지 분량, 학술 보고서 형식, 한국어
```

### 논문 요약
```
Role: 당신은 [분야] 전문 연구자입니다.
Context: 다음은 제가 검토 중인 논문의 초록입니다.
Instruction: 이 논문을 다음 기준으로 분석해주세요:
             1. 연구 목적 (1-2문장)
             2. 핵심 방법론 (2-3문장)
             3. 주요 결과 (3가지 bullet)
             4. 내 연구와의 연관성 (1-2문장)
Format: Markdown 형식, 한국어
```

### 데이터 분석 코드 생성
```
Role: 당신은 Python 데이터 분석 전문가입니다.
Context: 산업 센서 데이터(CSV 형식)를 분석해야 합니다.
         컬럼: 시간, 온도, 진동, 압력, 고장여부
Instruction: 예측유지보수를 위한 EDA(탐색적 데이터 분석) 코드를 작성해주세요.
Format: 주석이 포함된 Python 코드, 실행 가능한 형태
```

## 고급 프롬프트 기법

### Chain of Thought (CoT)
```
"단계별로 생각하면서 답변해주세요:
1단계: 문제 파악
2단계: 가능한 접근법 나열
3단계: 최선의 방법 선택 및 이유
4단계: 결론"
```

### Few-shot Prompting
```
"다음 예시를 참고하여 같은 형식으로 작성해주세요:

예시 입력: [예시 1]
예시 출력: [원하는 형식의 출력 1]

이제 다음을 처리해주세요: [실제 입력]"
```

## 주의사항
- 너무 긴 프롬프트는 오히려 성능 저하
- 모호한 표현 대신 구체적 수치/기준 제시
- 한 프롬프트에 한 가지 작업만 요청
