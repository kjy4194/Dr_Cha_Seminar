# 논문 리뷰: 비전 기반 반도체·디스플레이 검사 자동화

## 논문 정보

| 항목 | 내용 |
|------|------|
| **제목** | Deep Learning-Based Defect Detection for Semiconductor and Display Panel Inspection |
| **저널** | IEEE Transactions on Semiconductor Manufacturing |
| **발행연도** | 2024 |
| **키워드** | Defect Detection, CNN, YOLO, Automated Optical Inspection (AOI), Display Panel |

---

## 핵심 요약

반도체 및 디스플레이 패널 제조 공정에서 딥러닝 기반 자동 광학 검사(AOI) 시스템을 적용하여
기존 수동 검사 대비 검출 정확도 98.5%, 검사 속도 3배 향상을 달성한 연구.

---

## 주요 내용

### 연구 배경
- 디스플레이 패널 해상도 고도화로 미세 결함 검출 난이도 증가
- 기존 룰 기반(Rule-based) AOI 시스템의 한계: 다양한 결함 패턴 대응 어려움
- 딥러닝 모델의 도입으로 결함 유형 자동 분류 가능

### 결함 유형 분류
| 결함 유형 | 설명 | 검출 난이도 |
|----------|------|------------|
| 점결함 (Point Defect) | 픽셀 단위 밝기 이상 | 중간 |
| 선결함 (Line Defect) | 수평·수직 라인 불량 | 낮음 |
| 얼룩 (Mura) | 불균일 휘도 분포 | 높음 |
| 스크래치 (Scratch) | 표면 긁힘 | 중간 |
| 이물질 (Foreign Matter) | 먼지·파티클 부착 | 낮음 |

### 제안 방법론
1. **이미지 전처리**: 조명 보정, 노이즈 제거 (Gaussian Filter)
2. **모델 구조**: YOLOv8 기반 실시간 결함 감지
3. **데이터 증강**: Flip, Rotation, Brightness 조절로 학습 데이터 5배 확장
4. **후처리**: NMS(Non-Maximum Suppression)로 중복 박스 제거

### 주요 결과
| 지표 | 기존 방법 | 제안 방법 | 개선율 |
|------|----------|----------|--------|
| 검출 정확도 | 91.2% | 98.5% | +7.3%p |
| 처리 속도 | 0.8 FPS | 2.4 FPS | 3배 향상 |
| 오탐율(False Positive) | 12.3% | 3.1% | -9.2%p |

---

## 나의 연구에의 시사점

- YOLOv8 모델을 나노디스플레이 패널 검사에 적용 가능성 확인
- 데이터 증강 기법으로 소량 샘플 문제 해결 → 현재 과제의 데이터 부족 이슈에 직접 적용 가능
- 인라인 실시간 검사를 위해 GPU 기반 추론 서버 구성 필요
- Mura(얼룩) 결함은 별도 알고리즘 고도화 필요 (난이도 높음)
